How iResearchPro Works

Here we are going to show you of How iResearchPro in action. We hope you will get a clearer understanding of how iResearchPro works by looking at this example.

The purpose of this example is to show you how to use the app for a practical use. For this example, we are going to show you how you can use the app to research for ramen (Japanese noodle) for your culinary trip to Japan. We are going to show you how you can use the app to research for different ramen types and for the best ramen restaurants in Japan.

+ Researching for Ramen Types

For example, this may be one of your first few visits to Japan and you are not familiar with the different types of ramen. So we begin by researching the different types of ramen. We will show you how you use the Keyword List, Get Links and Notes feature to create keywords, get links and take notes about the research subject.

Task 1. Creating A Keyword List

Step 1. Add a New Keyword List

The 1st step we need to do is to create a Keyword List. To create a Keyword List, enter the name of the Keyword List in the Keyword List text box, Select 'Add' from the Command box and press 'Go'. As a guide, you may use the [item] - [attribute] as the format for the Keyword List name. So in this case, since we are searching for different types of ramen, we can use 'ramen - type' as the Keyword List name.

Step 2. Adding New Keywords

After we have created a new keyword list, the next step is to add new keywords to it. To do this, we are going to use the 'Get Google Suggest' feature. This feature allows you to generate a series of alternative keywords from Google based on the main keyword. To use this feature, first choose a main keyword, then please select the 'Generate Keyword From Google Suggest' command, enter the main keyword in the text box and press 'Go', a suggested list of keywords will be generated based on that keyword.

Step 3. Selecting Keywords

Next, we select the keywords that we want to use to get search engine results. To select keywords, please select the 'Select Keywords' command and select the number of keywords you want to choose.

Task 2. Getting Search Engine Results

Step 1. Select 'Get SER' command

After we have selected the keywords , we use the 'Get SER' feature to get to get search engine results. To do this, select 'Get Search Engine Results - From Selected Keywords' from the command box , select the appropriate attributes such as search engines, no of pages, countries and press 'Go'.
